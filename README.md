# Adaptive A/B Testing Platform

Дипломный проект по разработке информационной системы для адаптивного A/B тестирования.

## Структура проекта

- `traffic_generator/` - Генератор реалистичного пользовательского трафика.
- `ab_testing/` - Ядро системы A/B тестирования.
- `ml_models/` - Модели ML для оптимизации тестов.
- `web_interface/` - Веб-интерфейс для управления системой.
- `database/` - Работа с базой данных.
- `tests/` - Тесты.
- `scripts/` - Вспомогательные скрипты.

## Технологический стек

- Python, Pandas, NumPy, SciPy, Scikit-learn
- Flask
- PostgreSQL
- Docker
